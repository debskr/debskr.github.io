Feature Projects

Here are a few of my featured projects. Each project showcases my ability to deliver high-quality, interactive, and engaging experiences.

Node 9

Node 9 is a puzzle-based casual 2D game. The game mechanic involves connecting a total of nine nodes with a line without overlapping. The sequence in which the nodes are selected is generated randomly, making each level unique. The game features multilingual support and includes ad network implementation.

Kiloflight AR Game

Kilo Flight is an augmented reality game on a mission of 1971 with complete army camp setup .The basic premise has it takes into the real war environment of Bangladeshi Liberation war. Its a first person shooter game .The shooter will shoot from the sky level which will give them the feel of staying in a copter. As an added challenge, A Mortar consecutively targeting shooter to destroy. The Shooter will have to save his life from the missile and will try to destroy all the element to be the high scorer.

Sixer Game

Sixer is a 2.5D flick cricket game where players use finger flicks to score runs from the incoming ball. The challenge is that the ball swings randomly, so players must anticipate where it will turn. The game features branding placements and a team selection process during onboarding.

Beat The Speed Game

Speed brings to you “Beat The Speed” game. Run & Cross the distance in a forest. Beat The Speed features challenging physics-based gameplay and hours of replay value. Each level requires logic, skill and force to run. I have helped to develop the core gameplay and in development

The Detached Game

Detached is an addictive puzzle based platformer game for PC platforms developed by us. It is a story of a lost boy who needed to discover his kidnapped parents on the era of war. The calamity of war and troops are active for shoot to kill. Player has to find a way out to bring the boy back to his parents solving every level.

Anti-Tobacco Survey App

Developed a Hybrid Survey application for National Heart foundation. The project was aiming for conducting Anti-Tobacco Survey on Youth, Aided by Center for Disease Control (CDC) and funded by Melinda Gates foundation. The application comprised with features of conducting assisted and non-assisted surveys, audio voiceover-based questions, offline data management, online data syncing system, resume capability and backend dashboard.

Bugbite App

Bugbite App is a mobile application designed to showcase the company’s portfolio and services in AR, VR, Game and Interactive applications. Developed using a Unity Engine platform, the app incorporates native features during the onboarding process, including login, registration, password recovery, and login options via phone, email, and Facebook. The app also features a dashboard system with user profiles.Additionally, Bugbite App is integrated with a web-based admin panel, which allows for the management of user profiles, content, and the sending of customized app notifications

Warehouse Shelver App

Developed a warehouse-based product Shelver application for Top Brand Outlet (UK) Ecommerce brand for their warehouse operation. The application aims for helping users to shelve and pick products by scanning SKU Code without any human errors. The application receives and updates data through API from central warehouse management system.

Brahmanbaria AR App

I have developed an augmented reality-based mobile application that showcases seven significant places in Brahmanbaria district within a virtual 3D environment. This AR app features interactive audio voiceovers and visual elements, creating a unique experience. It includes dynamic elements such as vegetation simulation, a day-night cycle, and clouds that change over time. From an optimization standpoint, the solution employs refined models to ensure smooth performance. I was featured in the local newspaper “Prothom Alo” for our work on this augmented reality mobile application, which was created for the event “Titasher Tarunno” in 2019.

Navana Real-Estate AR App

I have worked on two significant projects for Navana real estate by developing advance Augmented reality application. The application showcases prime architectural projects in 3d for its clients to visualize. The application showcases a real-life visualization of entire architecture using crowd simulation, traffic simulation, cloud system, day/night cycle and many more features.

Ocubil AR App

Developed an Augmented Reality based application to introduce features of medicine through audio visual presentation for better understanding. This project was an initiative of Square Pharmaceuticals Ltd. I was responsible for Asset development, programming, UI/UX and deployment

PhysioTrack

Developed Physiotrack, a hybrid desktop application that helps to assist physiotherapy patients to do exercises more accurately. The solution works by detecting patients body joints using depth camera and does accurate angle calculations for exercises. This is a SaaS solution aimed for used as a medical equipment by orthopedic doctors and physiotherapy centers with proper assessment report.

Toshakhana VR

Developed an interactive 360 VR experience of Bangabhaban Toshakhana Museum for Oculus VR. This project was an initiative taken by the ICT division, Bangladesh. The application is a audio visual interactive experience from the first person POV of Tashakhana Museum.

Satellite VR

Developed The Bangabandhu Satellite VR, a virtual reality simulation I’ve developed for Digital Fair 2020, showcasing the cinematic journey of the Bangabandhu satellite. Deployed on HTC Vive, the simulation allows users to experience the mission from a first-person perspective, highlighting its significance to the nation. The cinematic VR journey, created entirely in 3D, features the launch of the Falcon rocket, the separation phase, and the deployment phase in an immersive view. Additionally, users learn about the satellite’s importance and its operational timeline.

Interactive Penalty Game

Developed an Interactive penalty shootout game that uses Kinect device to detect body movements to score a goal. The core mechanics of the game is user will be able to play the game on a large LED screen. A goalkeeper and a goalpost will be there on the screen with a football to shoot in center. The User has to do kick gesture in order to kick the ball on the screen to score a goal.

Interactive Wall

Developed an Interactive wall solution using Unity, that uses 3D Kinect depth camera to detect human body movements to reflect interaction data to large scale LED screens for visualizing information. The result being users can visualize any graphical data on screen based on their physical position on screen. This system was provided for Digital fair and Trade fair for local corporate MNCs.

Corona Awareness App

Developed a WebGL based Application for Corona Status dashboard during covid-19 pandemic. The application uses open-source API endpoints to show results in application developed using Unity Engine.

3D Commerce WebGL App

A Web based real-time 3D product viewer solution that works on any Ecommerce platform. It replaces traditional photo-based product showcase for ecommerce product page and provides a dynamic 3D product viewer solution for ecommerce customers. Also, Products can be managed dynamically through backend via API call. This solution helps customers to view the product in high quality to make purchase decision effectively.
